# React 简介与组件开发指南

## 🧠 什么是 React？

React 是一个用于构建用户界面的 JavaScript 库，由 Facebook 维护。它采用组件化开发方式，使 UI 更加可复用和易维护。

- 官方网站：[https://reactjs.org](https://reactjs.org)
- 核心特点：
  - 声明式编程
  - 组件化
  - 虚拟 DOM
  - 单向数据流

---

## ⚙️ 环境准备

安装 Node.js（推荐 LTS 版本），然后使用以下命令创建一个 React 项目：

```bash
npx create-react-app my-app
cd my-app
npm start
```

---

## 📦 React 组件基础

React 组件有两种：

### 1. 函数组件（Function Component）

```jsx
function Welcome(props) {
  return <h1>Hello, {props.name}!</h1>;
}
```

### 2. 类组件（Class Component）

```jsx
import React, { Component } from 'react';

class Welcome extends Component {
  render() {
    return <h1>Hello, {this.props.name}!</h1>;
  }
}
```

## 🧩 JSX 简介

JSX 是 JavaScript 的一种语法扩展，看起来像 HTML：

```jsx
const element = <h1>Hello, world!</h1>;
```

> 注意：JSX 最终会被编译为 `React.createElement(...)`

## 🪝 使用 React Hooks（仅限函数组件）

### useState 示例：

```jsx
import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);
  return (
    <div>
      <p>当前计数：{count}</p>
      <button onClick={() => setCount(count + 1)}>增加</button>
    </div>
  );
}
```

### useEffect 示例：

```jsx
import { useEffect } from 'react';

useEffect(() => {
  console.log('组件已挂载');
  return () => {
    console.log('组件将卸载');
  };
}, []);
```

## 🧱 Props 与 State 区别

| 特性   | Props                    | State                    |
|--------|--------------------------|--------------------------|
| 读写性 | 只读                     | 可读写                   |
| 来源   | 由父组件传入             | 组件内部初始化或更新     |
| 用途   | 通信、配置               | 控制组件行为、交互       |
